📋 PREREQUISITES (ONE TIME ONLY):

1. INSTALL PYTHON
   • Download: https://www.python.org/downloads/
   • Make sure to CHECK: "Add Python to PATH" during installation
   • Verify: Open CMD and type: python --version

2. INSTALL OLLAMA
   • Download: https://ollama.ai
   • Install and run: ollama serve
   • Download model: ollama pull mistral
   • Keep Ollama running in background

3. INSTALL PYTHON PACKAGES (ONE TIME)
   • Open CMD in C:\Users\awais folder
   • Run: pip install -r requirements.txt

🎮 HOW TO USE:

1. Emails arrive in your Gmail
2. Email Assistant automatically:
   • Reads the email
   • Analyzes what type it is (question, problem, thanks, etc.)
   • Generates a personalized reply
   • Creates a draft in Gmail
   • Marks email as READ
   • Saves to database

3. You see in "Drafts" page:
   • Original email
   • Generated reply
   • Buttons: Edit, Approve, Delete, Regenerate

4. Review & Approve:
   • Click "Approve" to send draft
   • Click "Edit" to change reply
   • Click "Delete" to discard
   • Click "Delete All" to clear all drafts

❓ FAQ:

Q: How do I stop the application?
A: Close both windows (Ollama and Email Assistant)
   No special shutdown needed

Q: Can I edit the replies?
A: Yes! Click "Edit" button in Drafts page
   Edit, save, then approve

Q: What if I don't like a reply?
A: Click "Delete" to discard
   Or "Delete All" to clear multiple

Q: How often does it check emails?
A: Every 60 seconds (configurable in config.json)

Q: Does it send emails automatically?
A: No! It only creates drafts
   You must click "Approve" to send

Q: Can I change the password?
A: Yes, edit config.json
   "password": "your_new_password"

Q: What if Ollama is too slow?
A: Normal - it uses free offline AI
   First reply takes 30-60 seconds
   CPU dependent
