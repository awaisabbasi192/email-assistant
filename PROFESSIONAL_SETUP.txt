📋 INSTALLATION STEPS:

PHASE 1: ONE-TIME SETUP (Do this once)

Step 1.1: Install Ollama
───────────────────────
  • Visit: https://ollama.ai
  • Click "Download"
  • Run installer
  • Complete installation
  • Restart computer (optional but recommended)

Step 1.2: Download Ollama Model
────────────────────────────────
  • Open CMD (Win + R → cmd → Enter)
  • Type: ollama pull mistral
  • Wait for download (5GB)
  • When done, you'll see "success"

Step 1.3: Install Python Packages
──────────────────────────────────
  • Open CMD in C:\Users\awais
  • Type: pip install -r requirements_desktop.txt
  • Wait for installation (2-3 minutes)
  • Look for "Successfully installed" message

Step 1.4: Build Desktop Application (Optional)
───────────────────────────────────────────────
  • This creates standalone .exe file
  • Not required if using Python method

  To build:
  • Open CMD in C:\Users\awais
  • Type: python build_installer.py
  • Wait for build completion
  • Creates: dist/EmailAssistantPro.exe

  This allows installation to Program Files
  (Desktop shortcuts will be created automatically)

🔧 TROUBLESHOOTING:

Problem: "Cannot connect to Ollama"
Solution:
  1. Make sure START_OLLAMA.bat is running
  2. Check message: "listening on 127.0.0.1:11434"
  3. If not, reinstall Ollama

Problem: "Application won't start"
Solution:
  1. Open CMD in C:\Users\awais
  2. Run: python desktop_app.py
  3. Check error messages
  4. Check email_assistant.log

Problem: "No emails processing"
Solution:
  1. Check Ollama running
  2. Check Gmail has unread emails
  3. Refresh browser (F5)
  4. Check logs

Problem: "Notifications not showing"
Solution:
  1. Check Windows notifications are enabled
  2. Check System settings
  3. May be disabled for certain apps

Problem: "Gmail auth failed"
Solution:
  1. Delete gmail_token.pickle
  2. Restart application
  3. Follow Google OAuth screen
  4. Grant permissions
