CUSTOMIZATION:
================================================================================

To change the primary color (electric blue):

1. Open: C:\Users\awais\web_ui\templates\base.html
2. Find: tailwind.config section
3. Change: primary-500: '#0ea5e9' to your color

Example colors:
- Purple: #a855f7
- Pink: #ec4899
- Green: #10b981
- Red: #ef4444
- Amber: #f59e0b

API ENDPOINTS (must exist in Flask app):
================================================================================

GET  /api/stats              - Dashboard statistics
GET  /api/draft/<id>         - Draft details
POST /api/draft/<id>/approve - Approve draft
GET  /api/email/<id>         - Email details
GET  /api/emails             - All emails
GET  /api/analytics          - Analytics data

If these endpoints don't exist, you'll need to add them to your Flask app.
