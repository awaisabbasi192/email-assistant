📊 DASHBOARD
  Purpose: Monitor everything at a glance
  Shows:
    • Total emails count
    • Processed emails count
    • Pending drafts (waiting for your review)
    • Processing rate (%)
    • Email categories chart
    • Service health (✓ or ✗)
  Auto-updates every 30 seconds

📬 EMAILS PAGE
  Purpose: See all emails in database
  Shows:
    • From (sender)
    • Subject
    • Category (personal, work, support, etc.)
    • Priority (urgent, high, normal, low)
    • Status (processed Y/N)
    • Date received
  Features:
    • Filter by category dropdown
    • Sort by any column
    • Pagination

✍️ DRAFTS PAGE ⭐ MOST IMPORTANT
  Purpose: Review and manage AI-generated replies
  Shows:
    • Original email on left
    • Generated reply on right
    • Confidence score (how good is response)
    • Model used (Mistral)
    • Creation date/time
  Actions:
    • ✓ Approve → Save to Gmail draft
    • ✎ Edit → Modify reply, then save
    • 🔄 Regenerate → New reply from Ollama
    • 🔗 Gmail → Open in Gmail
    • ✕ Delete → Remove this draft
  Status badges:
    • pending = waiting for review
    • approved = saved to Gmail
    • sent = you sent it
    • deleted = removed

🔑 KEY CONCEPTS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

DRAFT ≠ SENT
  Draft = Reply generated and stored, NOT YET SENT
  You review and modify drafts before sending
  This gives you full control!

WHY DRAFTS?
  • AI isn't perfect - you get to verify
  • Add personal touches
  • Catch errors before sending
  • Build trust with clients
  • Never send low-quality responses

CATEGORIES (Smart Classification)
  • personal = Friends, family
  • work = Colleagues, projects
  • support = Help requests
  • marketing = Promotions, newsletters
  • bills = Invoices, receipts
  • social = Social media notifications
  • newsletter = Subscriptions
  • urgent = Time-sensitive
  • spam = Unwanted emails

CONFIDENCE SCORE
  • How sure is Ollama about the reply?
  • 95% = Very confident, probably good
  • 75% = Confident, review recommended
  • 50% = Less confident, definitely review
  • Use as guide for how much to edit

PROCESSING RATE
  • What % of emails have been processed?
  • Goal: 100% (every email gets a draft)
  • Some emails might not need replies (system emails)
  • Aim for high processing rate
