📦 FIRST TIME SETUP (30 MINUTES)

Step 1: Install Ollama (One Time Only)
──────────────────────────────────────

What is Ollama?
  • Free local AI engine
  • Runs on your computer
  • No internet needed for processing
  • Privacy-friendly

Installation:
  1. Visit: https://ollama.ai
  2. Click "Download"
  3. Choose Windows version
  4. Run installer
  5. Follow instructions
  6. Restart computer (optional)

Verify:
  • Open CMD
  • Type: ollama --version
  • Should show version number


Step 2: Download Ollama Model (5-10 minutes)
─────────────────────────────────────────────

What is Model?
  • AI brain for generating replies
  • 5GB download
  • Downloaded once, used forever
  • Works offline

Installation:
  1. Open CMD
  2. Type: ollama pull mistral
  3. Wait for download
  4. When done: "success" message


Step 3: Install Python Packages (5 minutes)
─────────────────────────────────────────────

What are Packages?
  • Libraries needed to run app
  • Already listed in requirements file
  • One-command install

Installation:
  1. Open CMD in C:\Users\awais
  2. Type: pip install -r requirements_desktop.txt
  3. Wait for installation
  4. When done: "Successfully installed" message


Step 4: Build Executable (Optional, 5 minutes)
───────────────────────────────────────────────

Why?
  • Creates standalone .exe file
  • Can be shared with others
  • Professional distribution
  • Can be installed to Program Files

Installation:
  1. Open CMD in C:\Users\awais
  2. Type: python build_installer.py
  3. Wait for build
  4. Creates: dist/EmailAssistantPro.exe

❓ FAQ

Q: How do I stop the app?
A: Close the window or right-click tray → Quit

Q: Can I edit replies?
A: Yes! Click "Edit" button in Drafts

Q: Are emails sent automatically?
A: No! Only drafts created. You approve to send.

Q: How often does it check?
A: Every 60 seconds (configurable)

Q: Is my data safe?
A: Yes! Everything is local on your computer

Q: Can it work offline?
A: Email check needs internet (Gmail)
   But reply generation is offline (Ollama)

Q: Why is first reply slow?
A: Normal! First AI response takes 30-60 seconds
   Depends on your CPU

Q: Can I change the password?
A: Yes, edit config.json → password field

Q: How to uninstall?
A: Just delete the application folder
   No registry changes or cleanup needed

🆘 TROUBLESHOOTING

Issue: "Cannot connect to Ollama"
────────────────────────────────
Solution:
  1. Check START_OLLAMA.bat window
  2. Look for: "listening on 127.0.0.1:11434"
  3. If not there:
     a) Close CMD
     b) Double-click START_OLLAMA.bat again
     c) Wait for message
  4. Try running app again

Issue: "Gmail authentication error"
────────────────────────────────────
Solution:
  1. Find: gmail_token.pickle (in C:\Users\awais)
  2. Delete it
  3. Restart application
  4. Click to authorize Gmail
  5. Click "Allow" in browser

Issue: "No emails processing"
──────────────────────────────
Solution:
  1. Check Ollama running
  2. Check Gmail has unread emails
  3. Refresh browser (F5 or Ctrl+R)
  4. Wait 60 seconds for next check
  5. Check email_assistant.log for errors

Issue: "Application won't start"
─────────────────────────────────
Solution:
  1. Open CMD in C:\Users\awais
  2. Type: python desktop_app.py
  3. Read error messages
  4. Check email_assistant.log
  5. Make sure Ollama is running

Issue: "Notifications not showing"
──────────────────────────────────
Solution:
  1. Check Windows Settings
  2. Settings → System → Notifications
  3. Make sure notifications are enabled
  4. Check app notifications are on
